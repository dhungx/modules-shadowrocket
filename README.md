# Modules Shadowrocket
`modules-shadowrocket` là một tập hợp các module dành cho Shadowrocket, hiện bao gồm các module hỗ trợ cho YouTube và Spotify, giúp cải thiện trải nghiệm sử dụng ứng dụng.

## Các Module

- [YouTube Module](https://raw.githubusercontent.com/dhungx/modules-shadowrocket/main/youtube.module)
- [Spotify Module](https://github.com/dhungx/modules-shadowrocket/blob/main/spotify.module)
